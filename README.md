  - 1729128 - PHOENIX CONTACT.lbr: Conector PHOENIX

  - XC6206P332MR-G-TS.lbr: LDO

  - led_rojo.lbr: LED ROJO

- Esquematico.pdf: Esquemático EAGLE en formato pdf
- Esquematico.sch: ESquemático EAGLE en formato sch
